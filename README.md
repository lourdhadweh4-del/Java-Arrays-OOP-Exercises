# Java Arrays and OOP Exercises
This repository contains Java programming exercises focused on arrays and object-oriented programming concepts.

## Concepts Covered
- Arrays
- Object-Oriented Programming (OOP)
- Classes and objects
- Methods
- Loops
- Conditionals
- Problem solving
- Basic data manipulation

## Tools Used
- Java
- IntelliJ IDEA

## Purpose
To practice implementing arrays and applying object-oriented programming principles in Java.
These exercises helped strengthen understanding of how data is stored, accessed, and organized using classes and objects.
